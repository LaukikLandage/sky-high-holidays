# Sky High Holidays Website

Modern travel website with international & domestic packages, enquiry system, and scalable CMS-ready architecture.

## Features

- **Responsive UI**: Fully optimized for mobile, tablet, and desktop devices.
- **Destination Detail Pages**: Rich, interactive pages with itineraries and policies.
- **Enquiry Forms**: Seamless booking and contact flows with custom form validation.
- **Hero Video Background**: High-end cinematic transitions on the homepage.
- **SEO Optimized Content**: Structured semantic HTML and meta optimization.
- **Animated UI Elements**: Premium scrolling marquees and smooth page transitions using Framer Motion.

## Tech Stack

- **Frontend Framework**: React (via Vite)
- **Styling**: Tailwind CSS
- **Language**: TypeScript / JavaScript
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Future Scope

This project is architected for a future integration with a Headless CMS or custom backend to allow dynamic content management.
- **Admin Dashboard (CMS)**: Manage destinations, bookings, and content without code changes.
- **Backend Integration**: Implement Node.js/Express, Supabase, or Firebase.
- **Dynamic Content Management**: Upload images, update pricing, and change itineraries dynamically.
