Website Assets Structure Guide
==============================

This folder is organized to match the project's logic. You can drop your images into these folders to update the website.

Structure:
images/
├── 1_Destinations/
│   ├── International/
│   │   ├── Maldives/
│   │   │   ├── cover.jpg (Main card image - 4:3)
│   │   │   ├── gallery/ (Detailed gallery images - 16:9)
│   │   │   └── poster.jpg (Vertical poster - 4:5)
│   │   ├── Dubai/
│   │   └── ...
│   └── Domestic/
│       ├── Goa/
│       ├── Manali/
│       └── ...
└── About/
    └── founder.jpg (Founder profile - 4:5)

Usage:
1. Replace 'cover.jpg', 'founder.jpg', etc. with your own images.
2. Keep the filenames exactly as shown to avoid breaking links.
3. Recommended Ratios are mentioned in parentheses.
